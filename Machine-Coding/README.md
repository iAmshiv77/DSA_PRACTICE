# Machine Coding — Node.js / NestJS / React / MERN

## What is Machine Coding?
Write working code in 60–90 minutes. Interviewer evaluates:
- Code structure and cleanliness
- Error handling
- Edge cases
- API design
- Time & space complexity of core logic
- Database/query design

## Interview Tips
1. **Ask first**: Requirements, edge cases, scale. (5 min)
2. **Design first**: Database schema, API contract, component structure. (10 min)
3. **Code clean**: Meaningful names, small functions, no magic numbers.
4. **Handle errors**: Every external call has try/catch or error middleware.
5. **Think aloud**: Narrate your thought process.
6. **Test mentally**: Walk through a happy path + edge case.
